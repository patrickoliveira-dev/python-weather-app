# Weather App

Aplicação desenvolvida em Python utilizando a API Open-Meteo.

## Funcionalidades

- Consulta de clima atual
- Busca por cidade
- Geocoding automático
- Previsão para os próximos dias
- Histórico persistente em JSON
- Estrutura modular
- Programação Orientada a Objetos

## Tecnologias

- Python
- Requests
- Open-Meteo API
